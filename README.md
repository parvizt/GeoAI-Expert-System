# GeoAI Expert System 🌍🤖

Developed GeoAI Expert System, a Python/PyQt5-based intelligent desktop application for well data management, offline geological query handling, and ML-assisted analysis in petroleum and drilling workflows.
Designed the system with a scalable architecture for future ingestion of geological and drilling reports into SQLite/SQL Server, enabling optimized search, structured data storage, and intelligent reporting capabilities.



## ✨ Key Features
- **Rule-Based Chatbot:** An offline QA system for geological concepts.
- **Machine Learning Integration:** Predictive modeling for well data.
- **Data Simulation:** Asynchronous real-time data simulation using multi-threading.
- **Database Management:** Secure data storage with SQLite and AES-256 encryption.
- **Interactive UI:** Professional user interface built with PyQt5, featuring dynamic data flow diagrams.

## 🛠️ Technologies & Libraries
- **Language:** Python 3.9
- **GUI Framework:** PyQt5
- **Database:** SQLite
- **Data & ML:** Pandas, Scikit-learn (or relevant libraries)

## 🚀 How to Run
1. Clone the repository:
   `git clone https://github.com/parvizt/GeoAI-Expert-System.git`
2. Install dependencies:
   `pip install -r requirements.txt`
3. Run the application:
   `python Geoai_Expert.py`
